# 🎮 SenseKeyboard

**PS5 DualSense keyboard for Android TV**

A fully customizable on-screen keyboard designed for the PS5 DualSense controller on Android TV. Navigate with the D-pad, type with Cross, and customize everything from colors to animations.

## ✨ Features

- **Full DualSense support** — D-pad navigation, Cross to type, Triangle for space, Square for backspace, Circle to close
- **Shift** — L2 hold for temporary shift, double-tap L2 for caps lock, on-screen ⇧ button
- **Text selection** — L2 + L1/R1 to select text, with Copy & Paste buttons
- **Hold to repeat** — Hold X to repeat characters, Square to rapid-delete, L1/R1 for cursor movement
- **Word suggestions** — Autocomplete with learned vocabulary
- **6 theme presets** — PS5, Xbox, Steam, Minimal, Rounded, Retro (or fully custom)
- **Accent color** with brightness & saturation controls
- **Key color customization** — Primary, secondary, font color, all with opacity
- **4 navigation effects** — Wind particles, sliding border, ripple, ghost trail
- **3 click animations** — Fill, pop, flash
- **Multiple layouts** — US QWERTY, Danish QWERTY (more coming)
- **Numpad mode** — Calculator-style layout with operators
- **Symbol keyboard** — Full special character set
- **Number row** — Optional, shows !@#$%^&*() when shifted
- **Adjustable size & position** — Width, height, anchors, margins
- **Font customization** — 8 font families + size scaling
- **Live preview** — See your keyboard while customizing in settings

## 🎮 Controls

| Button | Action |
|--------|--------|
| **D-pad** | Navigate keys |
| **✕ Cross** | Type character (hold to repeat) |
| **△ Triangle** | Space |
| **□ Square** | Backspace (hold to repeat) |
| **○ Circle** | Close keyboard |
| **L1 / R1** | Move cursor (hold to repeat) |
| **L2 hold** | Shift (uppercase while held) |
| **L2 × 2** | Caps lock |
| **L2 + L1/R1** | Select text |
| **R2** | Enter |
| **L2 + R2** | New line |
| **L3** | Switch to symbols |
| **Options** | Done/Submit |

## 📱 Installation

1. Download the APK from [Releases](https://github.com/Zebratic/ps5-android-keyboard/releases/latest)
2. Install via ADB: `adb install SenseKeyboard-v1.0.0.apk`
3. Open the SenseKeyboard app on your TV
4. Follow the 3-step setup: Enable → Select → Type!

## 🛠️ Build from Source

```bash
git clone https://github.com/Zebratic/ps5-android-keyboard.git
cd ps5-android-keyboard
./gradlew assembleDebug
# APK at app/build/outputs/apk/debug/app-debug.apk
```
